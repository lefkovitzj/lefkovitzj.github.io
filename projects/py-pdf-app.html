The source code provides a GUI application for viewing and manipulation of PDF files. The features are included:
  • Page re-arrangement, creation, & deletion
  • Page rotation
  • Page insertion
  • Page deletion
  • Page watermarking
  • Page markup (red pen)
  • File watermarking
  • Text Extraction
  • Image Extraction
  • PDF Encryption
  • PDF Decryption
  • PDF Compression
  • PDF Signature
  • Metadata modification
  • Handle multiple PDFs at once
  • Automatic software version update checker
  • Software source code file tampering detection

The application also allows you to view the pages of a PDF, adjust the zoom, navigate between pages, use keyboard shortcuts, and save copies of a PDF.
