ACM Ollama Chat

ACM Ollama Chat

Project Description

ACM Ollama Chat is a custom wrapper utility to allow communication with a local deployment of open-source LLMs, specifically the Ollama 3:8b model. This project is intended for deployment on the University of Dayton ACM student chapter's Debian server using a Dockerized Ollama instance, a Python application and a bash script.

To confiugre the application on your server, follow the instructions provided in the project's README.md file.
